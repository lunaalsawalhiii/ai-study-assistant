## ðŸ”’ **Privacy & Limitations**

### **Demo Mode Limitations:**
- Data is stored **locally in browser** (not in database)
- Clears when user closes browser or clears cache
- No real authentication required
- Perfect for testing, NOT for production use

### **Real User Mode:**
If testers want to create real accounts:
1. Click "Create an account" instead of Demo Mode
2. Enter email, password, and preferred name
3. Data will be saved to Supabase database
4. Accounts persist across sessions
